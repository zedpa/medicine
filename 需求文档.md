## B. 本次新增需求（spec-002 · T1）

详见 `specs/spec-002-research-closed-loop.md`。该 spec 同时列出后续切片 T2–T6（韦恩图 PNG、STRING PPI 网络、GO/KEGG 富集、方法学&引用生成、网页可视化），按 P4「一次只推进一个 Stage」逐个过闸。

### 功能需求
- **FR-T1.1** `run_herb(query, disease=None)` / `analyze_herbs(herbs, disease?)` / CLI `--disease`：疾病参数可选，缺省时行为与 v1.0 完全一致（向后兼容）。
- **FR-T1.2** 疾病名→人类相关基因(gene symbol)+关联分数，按 `disease.score_min` 过滤；本地缓存(ns=disease)，可复现。
- **FR-T1.3** `intersect_targets(drug, disease)` 纯函数 → `intersection / drug_only / disease_only / counts`，结构直接喂韦恩图。
- **FR-T1.4** 提供疾病时 Excel 增 `疾病靶点`、`交集靶点` 两 sheet。
- **FR-T1.5** 口径集中于 `config/pipeline.yaml › disease`（source/score_min/max_targets/timeout），改动记 `CHANGELOG.md`。

### 口径（写入 config，遵守 P5）
- 疾病数据源：Open Targets Platform GraphQL（免密钥，0–1 关联强度）
- `disease.score_min = 0.1`（关联强度下限，可调）
- `disease.max_targets = 3000`

### 功能需求（T3 · PPI 网络）
- **FR-T3.1** 靶点优先用 药物×疾病 交集（非空时），否则全部药物靶点；<2 基因或 `ppi.enabled=false` 时跳过。
- **FR-T3.2** `StringClient.network(genes)` → STRING 互作边表（combined score 0–1，按 `ppi.score_min` 过滤），缓存可复现。
- **FR-T3.3** `hub_genes(edges)` 纯函数按度数找核心靶点；`to_graphml` 纯函数导出 Cytoscape 可读网络。
- **FR-T3.4** Excel 增 `Hub靶点`、`PPI边表`；同时写 `outputs/<药材>.graphml`。
- **FR-T3.5** 口径写入 `config/pipeline.yaml › ppi`（enabled/score_min=0.4/max_nodes=200/species/timeout）。

### 验收（T3，先写测试，红-绿）
- AC-8…AC-13 见 spec-002 §5.2，对应 `tests/test_ppi.py`（解析/hub/graphml/缓存不触网）、
  `tests/test_pipeline_ppi.py`（优先交集 / <2 跳过 / disabled）。
- 真实 API 冒烟：8 个高血压基因 → STRING 22 条边，hub AGT/ACE/AGTR1（RAAS 系统高度互联，与病理一致）。

### 功能需求（T4 · GO/KEGG 富集）
- **FR-T4.1** 靶点优先交集，否则全部药物靶点；<3 基因或 `enrichment.enabled=false` 跳过。
- **FR-T4.2** `EnrichrClient.enrich(genes, library)` → 富集表（BH 校正 adj_p≤`adj_p_max`、combined 降序、截 `top_n`），缓存可复现。
- **FR-T4.3** 多库（默认 GO_Biological_Process_2021 + KEGG_2021_Human）按库分组挂 `result.enrichment`。
- **FR-T4.4** Excel 增 `GO富集`、`KEGG富集`（term/p/adj_p/combined/n_overlap/overlap_genes）。
- **FR-T4.5** 口径写入 `config/pipeline.yaml › enrichment`（enabled/libraries/adj_p_max=0.05/top_n=20/timeout）。

### 验收（T4，先写测试，红-绿）
- AC-14…AC-17 见 spec-002 §6.3，对应 `tests/test_enrichment.py`（解析/过滤/排序/topn/缓存不触网）、
  `tests/test_pipeline_enrichment.py`（优先交集 / <3 跳过 / disabled）。
- 真实 API 冒烟：10 个高血压基因 → KEGG「Renin-angiotensin system / Renin secretion / 血管平滑肌收缩」、
  GO「systemic arterial blood pressure 调控」（adj_p<1e-7，与病理高度一致）。

### 功能需求（T5 · 方法学&引用生成）
